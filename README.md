# Proyecto Web: Club Atlético Ñeponeta

Este es un proyecto realizado para la materia de informática. El objetivo es crear un sitio web estático para un equipo de fútbol utilizando únicamente tecnologías básicas de desarrollo web: HTML y CSS.

## Estructura del Sitio

La página está dividida en tres secciones principales para organizar la información del club:

1. **Inicio (index.html):** Es la página principal donde se da la bienvenida a los seguidores del equipo. Incluye una breve introducción y una sección con las fotos de los últimos partidos y resultados destacados.
2. **Historia (historia.html):** En esta sección se cuenta el origen del equipo, cómo surgió la idea entre amigos y el significado de los colores del club.
3. **El Plantel (plantel.html):** Una página dedicada a presentar a los jugadores clave del equipo, con descripciones de sus posiciones en la cancha.

## Tecnologías Utilizadas

Para cumplir con los requisitos de la tarea, decidí mantener el código lo más simple y manual posible:

* **HTML:** Se utilizaron etiquetas semánticas básicas como `<header>`, `<nav>`, `<main>` y `<footer>` para darle estructura al contenido.
* **CSS:** Se trabajó con un archivo de estilos externo (`estilo.css`) para manejar el diseño. Se aplicaron estilos para:
  * Centrar el contenido en la pantalla de forma manual.
  * Personalizar los colores institucionales (azul y blanco).
  * Ajustar el tamaño de las imágenes y eliminar bordes en los archivos PNG para que el diseño sea más limpio.
  * Crear un menú de navegación horizontal sencillo.

## Cómo visualizar la página

Para ver el proyecto, simplemente hay que descargar los archivos y abrir el archivo `index.html` en cualquier navegador web.
